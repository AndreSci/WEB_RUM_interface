Запрос предназначен для создания заявки на выпуск пропуска гостю.

Вид запроса GET
команда - /DoCreateGuest

пример запроса:
127.0.0.1:8077/DoCreateGuest

Функция принимает json данные.
Пример:
{
    "user_id": логин из личного кабинета,
    "inn": инн компании,
    "id_remote": id, (Уникальный id заявки из ЛК)

    "FLastName": text,
    "FFirstName": text,
    "FMiddleName": text,
    "FCarNumber": text,
    "FDateFrom": дата по дням,
    "FDateTo": дата по дням,

}

Функция отвечает json
Пример ответа:
{
    "RESULT": "SUCCESS\WARNING\ERROR",
    "DESC": "описание при ошибке",
    "DATA": ""
}

Типы статусов заявки:
1 Пропуск заказан
2 Пропуск заказан, автомобиль в черном списке
3 Пропуск заблокирован
4 Пропуск выдан
5 Вход осуществлён
6 Выход осуществлён
