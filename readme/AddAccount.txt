Переместить п.е. от компании к сотруднику

запрос POST
/AddAccount

передаваемые значения в параметрах запроса

guid = FGUID сотрудника
units = (кол-во парковочных единиц)

пример:
127.0.0.1:8077/AddAccount?guid={41111111-D222-4333-A444-65555555555}&units=1


отвечает json:
{
    "RESULT": "SUCCESS\ERROR",
    "DESC": "",
    "DATA": ""
}
