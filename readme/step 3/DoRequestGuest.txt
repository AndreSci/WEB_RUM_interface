Запрос предназначен для создания заявки на выпуск пропуска гостю.

Вид запроса GET
команда - /DoRequestGuest

пример запроса:
127.0.0.1:8077/DoRequestGuest

Функция принимает json данные.
Пример:
{
    "user_id": логин из личного кабинета,
    "inn": инн компании,
    "id_remote": id, (уточнить где его брать в ЛК) (Уникальный для заявки, он же номер заявки)

    "FLastName": text,
    "FFirstName": text,
    "FMiddleName": text,
    "FCarNumber": text,
    "FDateFrom": дата по дням,
    "FDateTo": дата по дням,

}

Функция отвечает json
Пример ответа:
{
    "RESULT": "SUCCESS\ERROR",
    "DESC": "описание при ошибке",
    "DATA": ""
}
