Запрос предназначен для получения статуса гостя на компанию.

Вид запроса GET
команда - /GetGuestStatus

пример запроса:
127.0.0.1:8077/GetGuestStatus

Функция принимает json данные.
Пример:
{
    "user_id": id арендатора,
    "inn": инн компании
    "id_request": id заявки из запроса /GetGuestsList
}

Функция отвечает json
Пример ответа:
{
    "RESULT": "SUCCESS\ERROR",
    "DESC": "описание при ошибке",
    "DATA": ""
}
