Запрос предназначен для изменения статуса гостя.

Вид запроса GET
команда - /DoChangeStatus

пример запроса:
127.0.0.1:8077/DoChangeStatus

Функция принимает json данные.
Пример:
{
    "user_id": логин из личного кабинета,
    "inn": инн компании,
    "id_request": id заявки из запроса /GetGuestsList
}

Функция отвечает json
Пример ответа:
{
    "RESULT": "SUCCESS\ERROR",
    "DESC": "описание при ошибке",
    "DATA": ""
}
