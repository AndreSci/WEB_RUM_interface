запрос POST

/SetContacts

params - передаваемые ключи в параметрах запроса

guid = GUID сотрудника     (основной)
phone = номер телефона  (доп. параметр)
email = почта           (доп. параметр)

(должен быть минимум один из двух доп. параметров)


пример:
127.0.0.1:8077/SetContacts?fid={E800000-C222-4333-B444-3612312312312}&phone=some_phone2&email=test@test.org2


отвечает json:
{
    "RESULT": "SUCCESS\ERROR",
    "DESC": "",
    "DATA": данные пользователя с обновленной информацией из БД
}
