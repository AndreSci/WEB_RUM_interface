Запрос предназначен для блокировки пропуска гостя.

Вид запроса GET
команда - /DoBlockGuest

пример запроса:
127.0.0.1:8077/DoBlockGuest

Функция принимает json данные.
Пример:
{
    "user_id": id арендатора,
    "inn": инн компании,
    "id": id (Уникальный id заявки из ЛК)
}

Функция отвечает json
Пример ответа:
{
    "RESULT": "SUCCESS\ERROR\WARNING",
    "DESC": "описание при ошибке",
    "DATA": ""
}
