Запрос предназначен для получения список активных заявок гостей на компанию.

Вид запроса GET
команда - /GetGuestsStatus

пример запроса:
127.0.0.1:8077/GetGuestsStatus

Функция принимает json данные.
Пример:
{
    "user_id": логин из личного кабинета,
    "inn": инн компании
}

Функция отвечает json
Пример ответа:
{
    "RESULT": "SUCCESS\ERROR",
    "DESC": "описание при ошибке",
    "DATA": ""
}
