Запрос предназначен для теста номера автомобиля и проверки правильности его изменения.

Вид запроса GET
команда - /DoTestCarNumber

пример запроса:
127.0.0.1:8077/DoTestCarNumber

Функция принимает json данные.
Пример:
{
    "car_number": "а100-аа 777 rus"
}

Функция отвечает json
Пример ответа:
{
    "RESULT": "SUCCESS\ERROR",
    "DESC": "описание при ошибке",
    "DATA": "А100АА777"
}

Результат ответа должен быть:
1. Все буквы заглавные
2. Без пробелов
3. Убирает окончание rus/рус
4. Остаются только буквы и цифры
5. Латиница, которая входит в список букв Российского гос номера, меняется на кириллицу
