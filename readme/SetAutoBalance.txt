Добавить\убрать сотрудника в\из избранные

запрос GET
/SetAutoBalance

передаваемые значения в параметрах запроса

guid = FGUID сотрудника     (из запроса RequestEmployees)
units = значение больше нуля или 0 для отключения автобаланса

пример:
127.0.0.1:8077/SetAutoBalance?guid={41111111-D222-4333-A444-65555555555}&units=5


отвечает json:
{
    "RESULT": "SUCCESS\ERROR",
    "DESC": "",
    "DATA": ""
}
